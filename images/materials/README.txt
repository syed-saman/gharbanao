IMAGE NAMING GUIDE — Nirman Bazaar
===================================

Place your material images here using this naming pattern:

  BRICKS
    bricks1.jpg   bricks2.jpg   bricks3.jpg  (add more as needed)

  SAND
    sand1.jpg     sand2.jpg     sand3.jpg

  STONE CHIPS
    stonechips1.jpg   stonechips2.jpg   stonechips3.jpg

Supported formats : .jpg  .jpeg  .png  .webp
Recommended size  : 800×500 px or larger (landscape works best)
Max file size     : keep under 500 KB per image for fast loading

To add more images beyond 3, just add the file here AND add the
path to the matching "images" array in js/data.js.

After adding images, ask Claude to commit + push to GitHub.
